# Java Fundamentos – Programa Start (Capgemini)

Este repositório contém exercícios práticos desenvolvidos durante o módulo de
**Java – Fundamentos** do **Programa Start – Capgemini**.

Os exercícios foram originalmente desenvolvidos no **Eclipse IDE** e organizados
aqui de forma estruturada para fins de versionamento e portfólio.

## Conteúdo abordado
- Sintaxe básica da linguagem Java
- Estruturas condicionais
- Laços de repetição
- Arrays e ArrayList
- Métodos
- Lógica de programação orientada a objetos (introdução)

## Estrutura do repositório
Cada pasta representa um conjunto de exercícios, contendo apenas o diretório
`src` com os arquivos `.java`, mantendo o código limpo e independente de IDE.

## Objetivo
Consolidar os fundamentos da linguagem Java, demonstrando organização,
disciplina de estudo e boas práticas de versionamento com Git.
